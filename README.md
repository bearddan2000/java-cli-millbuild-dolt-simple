# java-cli-millbuild-dolt-simple

## Description
A java gradle build, that connects to dolt database.

## Tech stack
- java
- millbuild
  - 6.8.2

## Docker stack
- dolthub/dolt-sql-serverdb:latest
- nightscape/scala-mill

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
